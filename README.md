### Introduction

Croct is a headless CMS that lets you manage both static and dynamic content, perform A/B testing, and personalize experiences â€” all through a single SDK, without the need for external services or complex integrations..

### Installation

Run the following command to install the SDK:

```sh
npm install @croct/plug
```
Check our [getting started](https://croct.com/reference/sdk/javascript/installation) guide for more details.
